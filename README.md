Hey there, I created this weather app using react. Feel free to use it.
Just add your api key in .env file

.env file

REACT_APP_API_URL = 'https://api.openweathermap.org/data/2.5'
REACT_APP_API_KEY = // Add api key from open weather map
REACT_APP_ICON_URL = 'https://openweathermap.org/img/w'

Steps to add api key-
1. Go to https://home.openweathermap.org/users/sign_up and create your own account.
2. To get your API key, click on your username in the top right corner, and then on "my API keys".
